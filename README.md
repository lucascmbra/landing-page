## 🚀 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio técnico utilizando **Next.js + TypeScript + SCSS**, com foco em boas práticas de componentização, responsividade e performance.

---

## ▶️ Execução do projeto

### Clone o repositório

```bash
git clone https://github.com/lucascmbra/landing-page.git
```

### Instale as dependências

```bash
yarn install
```

### Inicie o projeto

```bash
yarn run dev
```

A aplicação estará disponível em: http://localhost:3000
